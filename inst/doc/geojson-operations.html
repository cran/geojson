The geojson package has functions to do basic operations on GeoJSON classes.

library("geojson")

First, letâ€™s make a GeoJSON object

x <- '{
 "type": "GeometryCollection",
 "geometries": [
   {
     "type": "Point",
     "coordinates": [100.0, 0.0]
   },
   {
     "type": "LineString",
     "coordinates": [ [101.0, 0.0], [102.0, 1.0] ]
   }
  ]
}'
(y <- geometrycollection(x))
#> <GeometryCollection> 
#>   geometries (n): 2

properties

Add properties

x <- '{ "type": "LineString", "coordinates": [ [100.0, 0.0], [101.0, 1.0] ]}'
res <- linestring(x) %>% feature() %>% properties_add(population = 1000)
res
#> <Feature> 
#>   type:  LineString 
#>   coordinates:  [[100,0],[101,1]]

Get a property

properties_get(res, property = 'population')
#> 1000

bbox

Add bbox - by default, if you donâ€™t pass a bbox into bbox_add() we attempt to calculate the bbox for you. You can also pass in your own bbox.

tt <- x %>% feature() %>% bbox_add()
tt
#> {
#>     "type": "Feature",
#>     "properties": {
#> 
#>     },
#>     "geometry": {

geojson in data.frameâ€™s

Itâ€™s really easy to put geojson class objects into data.frameâ€™s as well.

The ideal solution is to put them into tblâ€™s (see the tibble package)

Make a point

x <- '{ "type": "Point", "coordinates": [100.0, 0.0] }'
(pt <- point(x))
#> <Point> 
#>   coordinates:  [100,0]

Put the point into a tbl

library("tibble")
data_frame(a = 1:5, b = list(pt))
#> # A tibble: 5 x 2
#>       a b             
#>   <int> <list>        
#> 1     1 <S3: geopoint>
#> 2     2 <S3: geopoint>
#> 3     3 <S3: geopoint>
#> 4     4 <S3: geopoint>
#> 5     5 <S3: geopoint>

Another object, here a multilinestring

x <- '{ "type": "MultiLineString",
  "coordinates": [ [ [100.0, 0.0], [101.0, 1.0] ], [ [102.0, 2.0], [103.0, 3.0] ] ] }'
(mls <- multilinestring(x))
#> <MultiLineString> 
#>   no. lines:  2 
#>   no. nodes / line:  2, 2 
#>   coordinates:  [[[100,0],[101,1]],[[102,2],[103,3]]]

Put into a tbl

data_frame(a = 1:5, b = list(mls))
#> # A tibble: 5 x 2
#>       a b                       
#>   <int> <list>                  
#> 1     1 <S3: geomultilinestring>
#> 2     2 <S3: geomultilinestring>
#> 3     3 <S3: geomultilinestring>
#> 4     4 <S3: geomultilinestring>
#> 5     5 <S3: geomultilinestring>

Put the point and multilinestring into the same tbl

(df <- data_frame(a = 1:5, b = list(pt), c = list(mls)))
#> # A tibble: 5 x 3
#>       a b              c                       
#>   <int> <list>         <list>

#> 3     3 <S3: geopoint> <S3: geomultilinestring>
#> 4     4 <S3: geopoint> <S3: geomultilinestring>
#> 5     5 <S3: geopoint> <S3: geomultilinestring>
